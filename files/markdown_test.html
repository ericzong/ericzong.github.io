段落 vs. 换行

段落

对于 Markdown 而言，段落就是连续行上的文本。不同的段落前后以至少一个空行分隔。

注意，段落前后的空行不必是严格意义上的空行，只包含空格和制表符的行也视为空行。

换行

在 Markdown 中，键入回车并不能起到换行的效果，它造成的效果仅仅是相当于插入了一个空格。
要换行，需在行末插入至少 2 个空格，然后再键入回车。
